This project is a full-stack web application designed to predict the price of used cars based on user input. It utilizes a supervised machine learning model trained on the “Used Car Dataset” obtained from Kaggle. The application allows users to enter details such as car specifications, which are then processed by a Flask-based backend and passed to the trained model for prediction. The frontend is built using HTML and Tailwind CSS to provide a clean and responsive user interface, while MySQL is used for data storage and management. The project demonstrates an end-to-end machine learning workflow, including data preprocessing, model training, evaluation, and deployment within a full-stack environment.
